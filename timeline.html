Context & Starting Point

  • Had existing Zotero library with 91+ sources
  • Identified need for better organization
  • Had acquired core theory sources (Hall, Weick, Manning, Savickas)

Independent Work Before Collaboration

  • Accumulated 91+ sources in Zotero over time
  • Identified organizational problem independently
  • Brought clear need: better system for dissertation work

Collaborative Work

Developed 6-collection dissertation structure:

  1. Core Theory & Framework
  2. Population & Context
  3. Phenomenon/Intervention
  4. Methodology & Methods
  5. Emergent Themes (for later)
  6. Supplementary & Background

Created Obsidian vault architecture (7 folders):

  • 00 - Inbox
  • 01 - Literature Notes
  • 02 - Concept Notes
  • 03 - Framework Development
  • 04 - Data & Analysis
  • 05 - Writing Drafts
  • 06 - Advisor Meetings
  • 07 - Project Management

Ownership Statement

  • System design = Collaborative (researcher brought needs, AI brought patterns, built together)
  • Implementation = Researcher's independent work
  • Guide creation = Researcher's independent work
  • Problem-solving = Researcher's independent work
